# Frax Finance (frax-finance)

Frax Finance is a fractional-algorithmic stablecoin protocol featuring FRAX (a stablecoin), FXS (Frax Shares governance token), and Fraxswap — a TWAMM-based AMM DEX. The protocol is deployed across multiple chains and exposes its on-chain data via The Graph subgraphs.

## APIs

### Frax Finance GraphQL API

GraphQL subgraph for Fraxswap on Ethereum, exposing pairs, tokens, swaps, liquidity positions, TWAMM long-term orders, and protocol-level analytics via The Graph.

- **Human URL:** [https://docs.frax.finance](https://docs.frax.finance)
- **Base URL:** `https://gateway.thegraph.com/api/{api-key}/subgraphs/id/fraxswap-ethereum`
